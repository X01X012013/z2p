About This Chat Board

Please post your comment on the top except that you need to reply someone, this way, your comment will be seen.

If you edit your post, please make a new line and write "Edit: " so we know it is changed.

You do not need an account to chat; but if you have an account, please change your avatar to be different from everyone else, this will make distinguishing people easier.

Be respectful and try to post your comments in the right place.

"Report Issue" is only for issues, suggestions, and complaints related to this website. Game related issues (game bug, game account issues, etc) should be posted to the official forum:

http://forums.ageofempires.com/forum/53-age-of-empires-castle-siege/

If you are a lord and want to be moderator, please post your request in Alliance Management. You will need a Disqus account to be moderator.

You need to enable JavaScript to use this website (not an issue for most of the time).
